# bootstarter

Blank HTML/CSS/JS template for new projects based on Bootstrap (3.3.7) and enhanced with useful plugins:
* jQuery (1.12.4)
* jQuery mobile
* Select2 (4.0.3)
* Owl carousel (2.0.0 beta 3)
* Fancybox (2.1.5)
* Modernizr (2.8.3)
* Respond.js (1.4.2)

On the next release:
* animated mobile menu icon
